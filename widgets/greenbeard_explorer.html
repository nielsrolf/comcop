Gradient green-beard — smooth RGB look, proximity pairing

Every agent carries a heritable RGB look (its color, mutated by small Gaussian drift each birth) and a heritable tolerance τ. It cooperates with its co-players only if their average look-similarity is ≥ τ. Pairing is by proximity: each game recruits the agents nearest its location, so — because children spawn beside their parents — neighbors tend to be kin. Tag recognition and spatial structure stack. Runs auto-loop; click any agent to inspect it.

Configure
Per round, players in a game: mutual cooperate +1 each, mutual defect 0, one-sided defect gives the defector +2 and the cooperator −1 (matches games.py). Movement speed controls how much location matters: fast movement scrambles kin clusters toward well-mixed. Configuring anything stops the auto-loop.
