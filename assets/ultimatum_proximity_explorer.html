Ultimatum game — proximity pairing (neighbors ≈ kin)

Every agent carries two heritable genes: an offer (how generous it is as proposer) and a threshold (the minimum offer it accepts as responder). Pairing is by proximity: each game recruits the agents nearest its location, and since children spawn beside their parents, neighbors tend to be kin. Watch offers climb toward a near-even split rather than collapsing to the selfish corner. Runs auto-loop; click any agent to inspect it.

Configure
Each game, every player acts once as proposer against each co-player: if offer ≥ responder.threshold the pie (2.0) splits as (1−offer) to proposer and offer to responder, else both get 0 (matches UltimatumGame in games.py). Movement speed controls how much location matters: fast movement scrambles kin clusters toward well-mixed. Configuring anything stops the auto-loop.
